# Machine Learning for NLP

This folder contains my implementations, notes, and practical exercises while learning **Machine Learning techniques for Natural Language Processing (NLP)**.

These implementations are based on my learning from the **Complete Generative AI Course With Langchain and Huggingface** by Krish Naik.

---

## Topics Covered

### Text Preprocessing
- Tokenization
- Stemming
- Lemmatization
- Stopwords Removal

### NLP using NLTK
- Part of Speech (POS) Tagging
- Named Entity Recognition (NER)

### Feature Engineering
- One Hot Encoding (OHE)
- Bag of Words (BoW)
- N-Grams
- TF-IDF

### Word Embeddings
- Word2Vec
- CBOW
- Skip-Gram
- Average Word2Vec

## Technologies Used

- Python
- NumPy

## Learning Goals

- Understand text preprocessing techniques
- Learn feature engineering for text data
- Explore classical NLP algorithms
- Build a strong foundation before Deep Learning for NLP
